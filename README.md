# com.example.animalsappmvvm

Sample Android application displaying animals in the MVVM architecture.

Technologies and solutions used:

- Material Design
- Navigation
- LiveData
- Retrofit
- RxJava
- RxAndroid
- Glide
- Shared Preferences
- Palette
- Data Binding
- View Binding
- Dependency Injection using Dagger2
- Unit Testing using Mockito
